# Mission-Schule-ueberleben
Kennst du das?: Zu spät dran, Zettel muss zum Sekretariat, aber auf dem Gang steht überall jemand im Weg? Genau darum geht's hier.  Du läufst durch ne random generierte Schule, verlierst Nerven bei jeder Runde und musst hoffen dass du Kioske findest bevor Herr Krause dich erwischt. Pop-Quiz mittendrin nerven dich auch ohne Ende.
